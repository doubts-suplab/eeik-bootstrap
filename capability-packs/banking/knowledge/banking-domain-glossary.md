# Banking Domain Glossary

| Term | Banking Definition | NOT to be confused with |
|------|-------------------|------------------------|
| **Account** | A banking account holding a balance (current, savings, loan) | Not a user account in IAM |
| **Transaction** | A financial movement affecting an account balance | Not a database transaction (though related) |
| **Ledger** | The authoritative record of all financial transactions | Not a configuration file |
| **Debit** | Money leaving an account (reduces balance) | Not a debit card type |
| **Credit** | Money entering an account (increases balance) | Not creditworthiness |
| **Clearing** | The process of settling interbank transactions | Not cache clearing |
| **Settlement** | Final transfer of funds between banks | Not claim settlement |
| **AML** | Anti-Money Laundering — detecting/preventing financial crime | Not Amazon Machine Learning |
| **KYC** | Know Your Customer — identity verification process | Not a configuration option |
| **PSD2** | EU Payment Services Directive 2 — open banking regulation | Not a version number |
| **IBAN** | International Bank Account Number | Not an identifier type |
| **BIC/SWIFT** | Bank Identifier Code — international bank routing | Not a quick action |
| **Basel III** | International banking capital adequacy framework | Not a location |
| **NPL** | Non-Performing Loan — loan in default or near default | Not a programming language |
| **LTV** | Loan-to-Value ratio — loan amount ÷ asset value | Not a video format |
| **Chargeback** | Reversal of a card transaction, initiated by the cardholder | Not a system error |
