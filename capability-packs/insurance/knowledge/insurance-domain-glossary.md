# Insurance Domain Glossary

Use this glossary when building systems for insurance clients. Terms here override generic definitions.

| Term | Insurance Definition | NOT to be confused with |
|------|---------------------|------------------------|
| **Policy** | The insurance contract — defines coverage, exclusions, premium, insured | Not a Spring Security policy or IAM policy |
| **Premium** | The price paid for insurance coverage | Not a software subscription tier |
| **Claim** | A demand for payment under a policy following a covered event | Not a JWT claim |
| **Coverage** | The scope of protection provided by a policy | Not code coverage |
| **Excess** | Amount policyholder pays first (UK term); US = Deductible | Not an HTTP 5xx excess |
| **Underwriting** | Risk evaluation and acceptance/declination process | Not software version signing |
| **Bordereaux** | Detailed report of premiums or losses for reinsurance accounting | Not a data format |
| **Reserve** | Money set aside to pay future claims (outstanding claims reserve) | Not a database reserved word |
| **FNOL** | First Notice of Loss — first contact from policyholder reporting a claim | Always an event in the event catalog |
| **Loss Ratio** | Claims paid ÷ premiums earned × 100 | Not a ratio in code |
| **Solvency II** | EU insurance regulatory framework for capital adequacy | Not a version number |
| **FCA** | Financial Conduct Authority — UK insurance regulator | Not the Java FCA acronym |
| **Reinsurance** | Insurance bought by insurers to limit large-loss exposure | Not software redundancy |
| **Bordereaux Premium** | Premium bordereaux — list of risks ceded to reinsurer | Not a data list |
| **UW** | Underwriting (common abbreviation in insurance systems) | Not user warning |
