# /design-rag — Design a RAG Pipeline

Activate `rag-specialist` and design a retrieval-augmented generation pipeline.

## Usage
```
/design-rag "Claims policy document retrieval — users ask questions about their insurance policy"
```

## What This Command Does
1. Elicits document types, volume, update frequency, user query patterns
2. Recommends chunking strategy, embedding model, vector store
3. Designs ingestion pipeline and retrieval configuration
4. Defines evaluation metrics and baseline
5. Identifies governance requirements for the data being retrieved
