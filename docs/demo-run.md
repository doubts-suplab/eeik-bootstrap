## /estimate — Single Simple REST API Endpoint

**Feature:** A single simple REST API endpoint — `POST /orders`

### Assumptions

1. No auth, no payments, no external API calls — pure create + persist + 201 response
2. SQLAlchemy `Order` + `OrderItem` models built from scratch
3. Local dev environment already bootstrapped; no setup time included
4. GitHub Actions CI pipeline pre-exists; no pipeline authoring included
5. Authentication / authorisation out of scope for this slice
6. No domain events, message queues, or outbox pattern required
7. Single developer executing the full slice end-to-end
8. One PR review cycle assumed (20% overhead applied to implementation hours)
9. Pydantic v2 in use (default with FastAPI + Python 3.12 greenfield)
10. OpenTelemetry instrumentation expected but not a full observability integration task
